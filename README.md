TODOアプリ

React + Node.js で作成したシンプルなタスク管理アプリです。
タスクの追加・編集・削除・完了状態を管理でき、直感的に操作可能です。

使用技術

フロントエンド: React, CSS

バックエンド: Node.js, Express

データ保存: JSONファイル

機能

タスク追加

タスク編集

タスク削除

タスク完了切替（チェックボックス）

編集キャンセル・保存

セットアップ方法

コマンドプロンプトまたはターミナルで server フォルダへ移動

cd server
npm run dev


新しいターミナルタブを開き、client/src フォルダへ移動

cd client/src
npm start


ブラウザが自動で開き、アプリを利用可能
